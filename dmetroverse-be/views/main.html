🚇 DMetroVerse Backend API

This API serves as middle man for DMetroVerse App.

Available Endpoints:

📌 /main - API overview (this page)
🚉 /<origin>/<destination>/[type] - Fetch metro route data (type: 'least-distance' or 'minimum-interchange')
Example: /JIEE/SP/least-distance
🔍 /reddit-posts - Fetch latest Reddit posts about Delhi Metro
🕒 /test - Fetch metro data with a simulated future timestamp
